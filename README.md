## Project Overview

This project is a sustainable investment and portfolio management simulator.  
It combines financial analysis, ESG scoring, climate indicators and portfolio reporting.

The objective is to show how a portfolio manager or ESG analyst can integrate sustainability factors into investment decisions without ignoring risk and return.

## Key Features

- Multi-asset portfolio construction
- ESG scoring methodology
- Climate transition score
- Controversy risk monitoring
- Portfolio-level ESG dashboard
- Financial performance analysis
- Investment reporting output

## Tools Used

- Python
- Streamlit
- Pandas
- Excel
- Public financial and sustainability data
